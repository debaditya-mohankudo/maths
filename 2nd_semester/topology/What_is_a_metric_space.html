What is a Metric Space?

A metric space is a mathematical structure that allows us to measure distances between elements of a set.

Formal Definition:

A metric space is a pair (X, d) where:

  • X is a non-empty set.
  • d : X × X → ℝ is a function called a metric (or distance function).

The function d(x, y) must satisfy the following four properties for all x, y, z ∈ X:

  1. Non-negativity: d(x, y) ≥ 0
  2. Identity of Indiscernibles: d(x, y) = 0 if and only if x = y
  3. Symmetry: d(x, y) = d(y, x)
  4. Triangle Inequality: d(x, z) ≤ d(x, y) + d(y, z)

Example (Euclidean Space):

Let X = ℝ², the 2D Cartesian plane. Define the distance function as:

d(x, y) = √((x₁ - y₁)² + (x₂ - y₂)²)

This is the standard Euclidean distance. The pair (ℝ², d) is a metric space.

Application in Engineering:

In Electrical Engineering, you may encounter metrics when analyzing signal spaces, error analysis (like Mean Squared Error), or norms. All these concepts are rooted in the idea of metric spaces, where measuring "closeness" or "distance" is crucial.
